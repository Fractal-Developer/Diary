# Diary

> My Diary app built with Electron


## Fractal-Developer

```
$ npm install
```

### Run

```
$ bozon start
```

### Package and publish

```
$ bozon package windows --publish
```

Builds the app for OS X, Linux, and Windows, using [electron-builder](https://github.com/electron-userland/electron-builder).
